슬랙 Webhook 연결

연결 안 됨

슬랙 앱 → Incoming Webhooks → URL 복사 후 붙여넣기

Webhook URL 발급 방법

  • api.slack.com/apps 접속 → 앱 클릭
  • 왼쪽 메뉴 Incoming Webhooks 클릭
  • Activate Incoming Webhooks 토글 켜기
  • Add New Webhook to Workspace → 채널 선택 → 허용
  • 생성된 URL 복사 → 위 칸에 붙여넣기 → 저장
